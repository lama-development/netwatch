Network Settings

ⓘ Sets the maximum time (in seconds) to wait for a ping response before considering the host unreachable.

ⓘ Specifies the verbosity of the system logs, using levels such as DEBUG, INFO, WARN, and ERROR.

ⓘ Specifies the delay (in seconds) to wait after completing a full cycle of pinging all devices before starting the next cycle.

ⓘ Indicates the waiting time (in seconds) between consecutive retry attempts following a failed ping.

ⓘ Defines the total number of ping attempts to be made when a device appears offline before ceasing further retries.
